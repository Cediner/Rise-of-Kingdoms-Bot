# Rise of Kingdom Bot

### **Introduction**

Self using bot for Rise of Kingdom. 



### Requirements

- software

  - ADB (place under project directory: **<u>*.\adb*</u>**\)
  - tesseract (place under project directory:  ***<u>.\tesseract\</u>*** )

- libraries

  - opencv-python

  - pytesseract

  - numpy

  - pillow

  - pure-python-adb
